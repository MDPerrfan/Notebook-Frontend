# Notebook Website

The **Notebook Website** is a simple application where users can create an account, log in, and store their personal notes. It provides a clean and intuitive interface to manage notes securely.

## Features

- **User Authentication**:
  - Users can create an account and log in securely to manage their notes.
  - Passwords are encrypted for secure access.

- **Note Management**:
  - Users can create, edit, and delete notes.
  - Notes can be saved with titles for easy identification.
  
- **Responsive Design**:
  - The website is designed to be fully responsive, ensuring a seamless experience on both desktop and mobile devices.

## Installation

To set up the project locally, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/NotebookWebsite.git
   ```

2. Navigate to the project directory:

   ```bash
   cd NotebookWebsite
   ```

3. Install the necessary dependencies (assuming you’re using a package manager like `npm`):

   ```bash
   npm install
   ```

4. Start the project locally:

   ```bash
   npm start
   ```

The website should now be accessible on `http://localhost:3000`.

## Technologies Used

- **Frontend**: HTML, CSS, JavaScript (React)
- **Backend**: Node.js, Express 
- **Database**: MongoDB 
- **Authentication**:  JWT 
- **Styling**: Bootstrap(for responsive design)


## How to Use

1. **User Registration**:
   - Go to the registration page and create an account by providing your email and password.

2. **Log In**:
   - After creating an account, log in using your credentials to access your notes.

3. **Manage Notes**:
   - Once logged in, create, view, edit, and delete your personal notes.
